Coffi

El gestor de paquetes oficial para el lenguaje B++.

Instalación

Para instalar una nueva librería, simplemente ejecuta:

C:\> coffi install nombre_libreria

¡Funciona en cualquier terminal, hasta en tu JVdroid!
